class useful_lva_sdk.core.lva_graph_builder.LvaGraphBuilder(name)[source]

LVA graph builder.

Sample usage:

analysis = LvaGraphBuilder(“test-analysis”) .add_analyzer(GcsVideoSource(), “gcs_source”) .add_analyzer(GcsProtoSink(), “gcs_sink”) .add_analyzer(OccupancyCounting(), “oc”) .connect(“oc”, “gcs_sink”) .connect(“gcs_source”, “oc”) .build().analysis()
