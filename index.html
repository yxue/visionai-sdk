Welcome to Useful LVA SDK’s documentation!

This document lists APIs provided by useful_lva_sdk.

Installation

pip install useful-lva-sdk

Examples

 1 from useful_lva_sdk.core.lva_graph_builder import LvaGraphBuilder
 2 from useful_lva_sdk.core.operator import *
 3 from useful_lva_sdk.client.lva_client import LVAClient
 4
 5 analysis = LvaGraphBuilder("test-analysis") \
 6     .add_analyzer(GcsVideoSource(), "gcs_source") \
 7     .add_analyzer(GcsProtoSink(), "gcs_sink") \
 8     .add_analyzer(OccupancyCounting(), "oc") \
 9     .connect("oc", "gcs_sink") \
10     .connect("gcs_source", "oc") \
11     .build().analysis()
12
13 client = LVAClient(
14     project="PROJECT_ID",
15     location="LOCATION_ID",
16     cluster="CLUSTER_ID",
17 )
18 client.create_analysis(analysis.name, analysis)
